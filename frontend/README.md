# frontend

Next.js フロントエンドの配置先です。

- `src/app`: App Router
- `src/components`: 再利用 UI
- `src/features`: 機能単位の UI / ロジック
- `src/lib`: 共通関数やクライアント基盤
- `src/styles`: スタイル定義
- `public`: 静的アセット
- `tests`: フロントエンドテスト
- `.env.example`: フロントエンド用の環境変数サンプル
- `biome.json`: `biome` の設定
- `package.json`: `bun` で利用する依存管理と scripts
- `bun.lock`: `bun` 用のロックファイル
- `next.config.mjs`: Next.js 設定
- `Dockerfile`: フロントエンド用コンテナ定義
